# 🔤 React 使用自定义字体 ttf

## 📥 使用`@font-face`
下载字体....

🔍 [google-font](https://fonts.google.com/?categoryFilters=Sans+Serif:%2FSans%2FHumanist)

📝 在`index.css`中添加
```css
@font-face{
  font-family: 'pingfang';
  src:url('./assets/pingfang.ttf');
}
```

## 🎨 在`font-family`中添加

```css
font-family: 'pingfang', 'hk';
```
