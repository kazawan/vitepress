添加了一个新的功能，允许用户在点击标签时，直接跳转到对应的分类页面，而不是仅仅显示标签内容。这个功能是通过在 `CategoryTagView` 组件中添加一个新的 prop `categoryOrTag` 来实现的。
这个 prop 可以是一个字符串，表示要跳转到的分类名称。用户可以在点击标签时，直接跳转到对应的分类页面，而不是仅仅显示标签内容。
